# ZENIHOST - Bot Hosting Platform

Simple, Secure, and Powerful Bot Hosting System

## Features
- ✅ Simple username/password login (No Firebase/Google)
- ✅ Upload .py or .zip bot files
- ✅ Auto-install requirements.txt
- ✅ Start/Stop/Delete bots
- ✅ Real-time logs viewing
- ✅ Admin control panel
- ✅ User management (ban/unban)
- ✅ Plan system (Free/VIP/Premium)
- ✅ Telegram notifications

## Installation

# 2. Run the app
